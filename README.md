## 🚀 Como Usar

### Deploy Local para Teste

```bash
# Opção 1: Usando Python
python3 -m http.server 8000
# Acesse: http://localhost:8000

# Opção 2: Usando Node.js
npm run dev
# Acesse: http://localhost:8000
```

### Deploy na Vercel

```bash
# Instalar Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```
